F.A.Q.

What is Windows 7 Innovators Edition (Win7IE)?

Windows 7 Innovators Edition is a free modification of Windows 7 Professional.

Why Win7IE?

Win7IE includes many features and many applications built in such as Java JRE and Visual Studio Code.

Is it legal?

It is completely legal. Distribution of modified Windows is completely legal as long as it doesn't contain any closed-source software, doesn't bypass activation, and doesn't claim to own Windows.

How to install

Materials Needed

1.) A Windows 7 Professional Key

2.) A DVD or USB Thumb Stick with at least 8GBs of storage

3.) A computer (obviously)

4.) The Win7IE ISO

Preparing installation media

USB

Step 1.) Download Rufus

Step 2.) Open Rufus and select the ISO

Step 3.) Press "Start"


DVD

Step 1.) Insert the ISO into your computer

Step 2.) Navigate to the ISO

Step 3.) Right click it and select "Windows Disk Image Burner" or any other burner software you use.

Note: If you're using Linux, we recommend using GnomeBaker

Installation

Step 1.) Insert your installation media into the computer you want to install on.

Step 2.) Boot into the installation media. If it asks to press a button to boot DVD, do so. If it doesn't boot into the media, go to the bootloader on your PC and select the DVD. It is usualy ESC, F1, F2, F8, F10, or F12.

Step 3.) Follow the instructions on screen.

Note: Remember to read the EULA which is installed in the installation process.
